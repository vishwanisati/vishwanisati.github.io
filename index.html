2

Smart PPE for Occupational Hazard Prevention

Artificial intelligence

Description

Wearable technologies have begun to play animportant role in the workplace. This project presents a Smart PPE (Personal Protective Equipment) solution which employs a sensor network located on a helmet and a belt to monitor the state of the worker and the environment. The model implements continuous risk monitoring biometrics of the worker, detects the external impact, shock, luminosity, gases, temperature of the environment and provides real-time recommendations to workers.
